# Notas de Pesquisa

Use esta pasta para guardar insumos de vídeos, artigos e pesquisas que vão alimentar as aulas.

## Padrão recomendado
Criar um arquivo por habilidade, por exemplo:
- `notes/H18_linguagens.md`
- `notes/H21_matematica.md`

## Estrutura mínima
- Ideias-chave (resumo do material)
- Exemplos de contexto ENEM
- Pegadinhas comuns
- Frases úteis/repertório
